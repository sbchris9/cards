# CardsApp

A drag and drop webapp, where the user can take notes on cards, and sort them into tables.

## Features 

* Typescript monorepo with shared code between FE - BE
* GraphQL API
* JWT authentication (access token, refresh token)

## Dependencies

* nodejs v12
* docker
* yarn

```
yarn install
yarn start
```
